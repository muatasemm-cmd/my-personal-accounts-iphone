نسخة iPhone-first PWA
=====================

هذه النسخة مخصصة لتكون بداية تشغيل الآيفون بدون Mac.

ماذا نقلنا من نسخة ويندوز:
- الرئيسية: ملخص اليوم والشهر + آخر العمليات + الالتزامات القريبة
- الدخل والمصاريف: إدخال سريع + سجل موحد
- الأشخاص والديون: حركة دين/دفعة + كشف مبسط
- الخطة الشهرية: سقف الصرف + هدف التوفير + التهيئة
- النسخ والخصوصية: تصدير/استيراد JSON + قفل + إخفاء أرقام

المجلد:
D:\PersonalAccounts\PersonalAccounts\PersonalAccounts\iphone-pwa-v1

الملفات الأساسية:
- index.html
- css\app.css
- js\app.js
- manifest.webmanifest
- service-worker.js

ما الذي ما زال قادمًا:
- رسوم بيانية خفيفة
- بحث أسرع
- التزامات متكررة
- تصنيف مصاريف أذكى
- تدقيق أفضل على الإدخال
- تهيئة نهائية لرفعها كرابط PWA للآيفون
